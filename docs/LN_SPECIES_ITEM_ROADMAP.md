# LN 种族物品推进清单

## 规则边界

- 种族特性和灵弦分层处理。前者是生理、代谢、感官或种族底座；后者是可装配、可成长的能力权限。
- 商品要区分“化学生理产物”和“灵能凝质”。以魅魔为例：`魅毒` 是体液里的化学分子，`魅气` 是灵能转化后的气体/凝质，两者不能混写。
- 这里的“候选物品”分两类：
  - `显式命名`：种族条目里已经直接写出名字，可以优先落地。
  - `条目推导`：条目明确描述了稳定产物、素材或黑市流通品，但名字还需要统一命名。
