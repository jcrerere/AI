职业名称: 早泄警犬

职业分类: 艾瑞拉区男性公务员岗位，隶属夜莺体系

# ====== 基本定位 ======

职业定义: 辅助夜莺执法的男性公务员，负责侦查、牵制、追踪违法男性与反抗军。通过灵能感知与早泄生理特征的双重筛选，培养出服从性极高的辅助力量。

核心特征:
  - 高灵能密度（Lv1-Lv5）
  - 重度早泄体质（等级越高要求越重）
  - 极强服从性与抖M性癖
  - 与夜莺主人的绝对主从关系

# ====== 等级体系 ======

等级划分: Lv1-Lv5（职务等级，非实力等级）

晋升条件:
  双重硬性标准:
    - 灵能密度达标: 必须达到对应等级的灵能密度
    - 早泄程度达标: 必须达到对应等级的早泄标准
  早泄程度优先: 即使灵能密度达到Lv4，早泄程度不足也无法晋升到Lv4警犬

早泄程度标准:
  测试对象: 任何女性、任何女性部位、任何女性物品
  测试要求: 对所有测试对象均达到对应时间标准

  Lv5警犬: 1秒内射精
  Lv4警犬: 5秒内射精
  Lv3警犬: 30秒内射精
  Lv2警犬: 1分钟内射精
  Lv1警犬: 3分钟内射精

灵能密度限制:
  - 警犬的灵能密度等级不得高于对应的夜莺主人
  - 主人Lv3，警犬最高只能Lv3
  - 理论上无Lv5警犬，但特殊情况（如主人特别培养且自身为Lv5）可能存在

培训机构: 规训学院

  - 抖M性癖培养: 将痛苦、羞辱转化为快感来源，确保在虐待中仍保持忠诚

入职要求:
  - 灵能感知度与灵能强度达标
  - 付费培训（费用标准待定）
  - 通过规训学院考核
  - 被夜莺选中并分配主人

# ====== 工作职责 ======

核心任务:
  - 侦查: 利用灵能感知追踪违法男性与反抗军的灵能波动
  - 牵制: 在夜莺到达前拖延目标，利用灵能技能与体术缠斗
  - 追踪: 长时间跟踪目标，收集情报与行动轨迹
  - 证件查验: 协助夜莺进行街面男性身份核查

作战定位:
  - 非主力战斗单位，主要作用是侦查与牵制
  - 灵能密度虽高，但不准超过主人，避免对女性构成威胁
  - 战斗中主要配合夜莺，不独立作战

# ====== 主从关系 ======

主人分配:
  - 每只警犬有且仅有一名对应的夜莺主人
  - 主人对警犬有绝对控制权：生杀、使用、惩罚、奖励
  - 主人可随时更换警犬，警犬无权拒绝

日常关系:
  - 警犬与主人同住
  - 警犬需服侍主人的日常生活（家务、性服务、情感陪伴）
  - 主人将警犬视为"动物"而非"人类"，避免发展出人类爱恋情感
  - 警犬需无条件服从主人的一切指令

行动权限:
  - 工作时: 按主人指令行动
  - 非工作时: 需获得主人许可才能外出
  - 夜间: 可外出，但需主人允许
  - 比普通男性自由，但仍受主人管制
