<WORLD_机关_淬灵署>
机关名称: 淬灵署

机关分类: 血玫瑰工业与财政机关，负责灵能币铸造、工业产能与工人体系监管

# ====== 基本定位 ======

职能定位: 淬灵署是艾瑞拉帝国真正的工业底盘与财政阀门，控制淬灵区的压缩中枢、分解回收厂、炼线试验塔与桥口结算站。它决定灵能币如何被铸造、分解、登记与跨区流通，也决定底层男性劳工如何被分配、消耗与回收。

核心职能:
  - 灵能币铸造: 负责低级币向高级币压缩、高级币向低级币拆解与批次登记
  - 工业配额: 为淬灵区工厂、聚集塔与高压生产线设定产能目标
  - 劳工管理: 监管底层男性劳工、谐振师与工位编制，维持“生产即服从”秩序
  - 桥口结算: 审核跨区物资、币流与工业批次，记录官方兑换轨迹
  - 财政命脉: 为血玫瑰提供稳定币制与工业税基，是帝国财政的真实支点

# ====== 核心设施 ======

关键设施:
  - 压缩中枢: 低级灵能币向高级灵能币压缩的核心设施
  - 分解回收厂: 高级灵能币拆解、回流与工业回收节点
  - 炼线试验塔: 优化压缩与分解工艺、降低损耗率的技术设施
  - 桥口结算站: 所有跨区工业批次、币流与特殊工位调拨的登记关口

内部岗位:
  - 登记官: 审核批次、币流与跨区调拨记录
  - 炉线主管: 负责高压工厂与铸币线的生产纪律
  - 谐振师管理员: 监管谐振师与灵能恢复效率
  - 回收审计员: 处理损耗、回收、拆币与灰产渗漏问题

# ====== 对外关系 ======

与血薇:
  - 淬灵署为血玫瑰提供财政底盘，是首都法统得以维持的物质前提

与夜莺:
  - 夜莺负责抓人、押运与基层控制，淬灵署负责把“可用男性”转为稳定产能

与真理部:
  - 真理部定义“劳动即荣耀”的秩序叙事，淬灵署则把这套叙事落实为工业配额和损耗标准

与黑玫瑰:
  - 黑玫瑰长期试图绕开淬灵署，建立独立币制与工业链，因此淬灵署始终将诺丝区视为高风险渗透源

# ====== 玩家接触面 ======

常见接触方式:
  - 在淬灵区工厂、结算站、压缩中枢或回收线工作、被押运或被审计
  - 因大额兑换、异常币流、走私批次或损耗异常被结算站盯上
  - 接触谐振师岗位、配额制度、劳工工位与工业惩戒
  - 通过黑市绕过淬灵署的官方汇率与批次登记

叙事用途:
  - 把“帝国很强”落到可见的工业秩序、压迫劳动与财政机器上
  - 为工厂潜入、批次调包、走私、伪造登记、劫持币流提供机构锚点
  - 让玩家理解为什么淬灵区既是财富来源，也是最压抑的地区之一
