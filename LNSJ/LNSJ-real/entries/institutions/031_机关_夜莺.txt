机关名称: 夜莺

机关分类: 血玫瑰执法机关，负责治安与税收

# ====== 基本定位 ======

职能定位: 艾瑞拉区与汐屿区的治安执法与税收征管机构，血玫瑰在基层的权力延伸，维持女尊秩序的暴力工具

核心权限:
  - 治安执法: 街面巡逻、证件查验、违法处置
  - 税收征管: 每月向男性征收灵能币，强制执行
  - 暴力裁量: 对违法男性有当场处决权
  - 早泄警犬管理: 拥有、使用、处置早泄警犬

# ====== 等级体系 ======

等级划分: Lv1-Lv5

等级标准:
  双重因素决定:
    - 自身灵能密度等级
    - 所属早泄警犬的等级与数量

  配置要求:
    - Lv1夜莺: 无早泄警犬
    - Lv2夜莺: 必须拥有至少1只Lv1早泄警犬
    - Lv3夜莺: 必须拥有至少1只Lv2早泄警犬（可同时拥有多只低等级警犬）
    - Lv4夜莺: 必须拥有至少1只Lv3早泄警犬
    - Lv5夜莺: 必须拥有至少1只Lv4早泄警犬（理论上极少，因Lv5警犬罕见）

  晋升逻辑:
    - 灵能密度提升是基础
    - 培养出对应等级的警犬是必要条件
    - 部分夜莺为避免培养麻烦，会在晋升后更换警犬（淘汰旧警犬，申请已训练好的高等级警犬）

警犬配置:
  - 早泄警犬总数多于夜莺
  - 部分警犬处于"无主"状态（主人阵亡、被淘汰、或作为储备）
  - 无主警犬由夜莺总部统一管理，作为替补或临时调配

# ====== 工作方式 ======

日常职责:
  街面巡逻: 查验男性证件、监控Beta芯片信号、发现违法行为
  税收征管: 每月上门收取灵能币，核对工作记录与Beta芯片交易数据
  违法处置: 抓捕、审讯、处决或移送规训学院

区域差异:
  艾瑞拉区夜莺:
    - 权力绝对: 对男性拥有生杀大权，男性视夜莺为神
    - 敲诈策略: 有钱男性是长期提款机，压榨但不做绝（避免饿死或被举报）
    - 风险控制: 做得太狠会被举报到审判庭，需花钱抹案底，有案底无法在本区域继续执法，影响人际关系
    - 执法环境: 需应对反抗军、违法男性、税收任务，工作繁忙且有实权

  汐屿区夜莺分部:
    - 清水岗位: 无反抗军、无穷苦违法男性、无男奴公寓税收任务
    - 主要职责: 要员护卫、证件查验、维持表面秩序
    - 消遣方式: 因工作无聊，喜欢找有钱男性的茬作为娱乐
    - 行为边界: 不做得太出格，但也不放过，利用执法权玩弄男性消磨时间
    - 社会认知: 被视为"养老岗位"，夜莺们都想调回艾瑞拉区

装备与制服:
  标准装备: 项圈、手铐、鞭子、个性化武器（根据个人风格选择）
  制服: 情趣警察服风格，兼具威慑性与性暗示

# ====== 组织结构 ======

总部: 位于艾瑞拉区，负责指挥协调、审讯、武器库管理
分部: 汐屿区设有分部，职责相对轻松，被视为清水岗位

内部关系:
  - 等级森严，高等级夜莺对低等级有绝对命令权
  - 警犬归个人所有，不归组织统一调配（除无主警犬）
  - 内部竞争激烈，晋升需要实力与警犬培养能力

# ====== 社会地位 ======

在女性眼中:

  - 在血玫瑰体系中属于基层执行机关
  - 权贵女性可以命令夜莺，但普通女性需服从夜莺执法

在男性眼中:

  汐屿区男性: 警惕与无奈，知道夜莺会找茬但无力反抗
