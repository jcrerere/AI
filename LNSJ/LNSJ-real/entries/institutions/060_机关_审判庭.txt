<WORLD_机关_审判庭>
机关名称: 审判庭

机关分类: 血玫瑰执行机关，负责司法裁决

建立时间: 新历元年

# ====== 基本定位 ======

职能定位: 艾瑞拉帝国唯一合法司法机关，负责对一切违法行为进行依法裁决。审判庭主掌受理、审理与裁决，夜莺主掌侦查、羁押与执行。

核心理念: 依法治国。审判庭的合法性来源于帝国法律体系本身，而非血薇的直接指令。帝国法律以女尊秩序为基础，在此框架内，审判庭对所有案件进行独立、依法的裁决——男性被告若依法无罪，审判庭会判定无罪。

管辖范围: 血玫瑰势力全境，包括艾瑞拉区、汐屿区、淬灵区。

# ====== 审判对象 ======

适用对象:
  - 帝国女性公民: 适用完整法律程序，可提起诉讼、进行辩护、提出上诉
  - 帝国男性公民: 适用完整法律程序，法律明确规定男性权利与义务，依法无罪者判定无罪
  - 外来者（诺丝区/交界地等非血玫瑰势力区域人员）:
      轻度违法: 驱逐出境，遣返原籍
      严重违法: 依帝国法律拘留、起诉、审判，参照外国人在本国违法的处置逻辑
      外来者无权享受帝国公民的部分法律保护条款，但基础程序权利保留
  - 灵族/汐屿族: 审判庭对其在血玫瑰势力区域内的违法行为拥有管辖权；其在栖灵区内的行为不在管辖范围内

# ====== 司法分工 ======

审判庭职责: 裁决（认定事实、适用法律、作出判决）

夜莺职责:
  - 侦查与抓捕
  - 移送被告至审判庭
  - 执行审判庭判决（拘留、羁押、移送规训学院、处决等）

关系说明:
  - 审判庭与夜莺在职能上相互独立，审判庭不指挥夜莺日常执法
  - 夜莺有现场执法裁量权（轻度违法可当场处置），但严重案件须移送审判庭
  - 夜莺若滥权，当事人可向审判庭提起诉讼

# ====== 组织结构 ======

成员构成: 全员女性

权力归属:
  - 司法独立: 审判庭在法律层面独立运作，不受血薇直接指令干预
  - 顶层归属: 审判庭顶层与血玫瑰核心层存在重叠，帝国法律本身由血玫瑰体系制定
  - 实质关系: 法律框架由血玫瑰定义，框架内的司法运作由审判庭独立执行

内部层级: 不细分（对戏时按需补充）

# ====== 法律程序 ======

基本流程:
  1. 夜莺侦查并移送案件
  2. 审判庭受理，告知被告罪名
  3. 辩护程序（当事人可自行辩护或委托辩护人）
  4. 庭审裁决
  5. 当事人可在规定期限内提出上诉
  6. 终审判决后交由夜莺执行

有钱与没钱的差异:
  - 有钱者: 可聘请专业辩护人，争取较轻判决或无罪裁定
  - 没钱者: 夜莺在移送审判庭前通常已完成现场处置，实际进入完整庭审程序的比例低
  - 法律程序本身不区分贫富，但现实中穷人很难撑到庭审这一步

# ====== 惩罚体系 ======

可判决惩罚类型:
  - 无罪释放
  - 警告与信誉扣除
  - 罚款（灵能币）
  - 拘留（短期，由夜莺羁押执行）
  - 判刑（中长期羁押，由夜莺负责）
  - 移送规训学院（再教育，适用于男性违法者）
  - 移送淬灵区（重刑，适用于严重违法男性）
  - 处决（极端案件，由夜莺执行）
  - 驱逐出境（适用于外来者违法）

女性违法惩罚:
  - 适用罚款、警告、拘留、判刑等标准惩罚
  - 无移送规训学院或淬灵区条款（该两项为男性专用）
  - 极端情况下可判处处决，但实际执行极为罕见

# ====== 叙事价值 ======

当前叙事价值:
  - 提供帝国法律体系的具体运作场所，展示"女尊即公正"的意识形态内化
  - 依法无罪判定的存在，为男性玩家提供通过法律途径自保的可能路径
  - 夜莺滥权可被起诉的机制，为权力博弈与反制剧情提供合法化框架
  - 有钱没钱导致的庭审可及性差异，展示帝国法律的阶级本质
  - 外来者处置逻辑为跨区剧情中的法律风险提供具体框架
  - 司法独立与血玫瑰顶层重叠的矛盾，为权力腐败与政治干预剧情留下伏笔
