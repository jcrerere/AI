<WORLD_机关_血薇>
机关名称: 血薇

机关分类: 血玫瑰最高权力机关，负责政务协调与帝国决策

建立时间: 新历元年

# ====== 基本定位 ======

职能定位: 帝国最高行政机关，负责协调血玫瑰体系下所有机关的运作与利益分配，是帝国政令的最终来源与执行裁定者。

权力性质:
  - 血薇在血玫瑰体系内是理论上的上级机关，对其余七大机关拥有政令下达权
  - 其余机关并非无条件盲从，血薇的实际话语权建立在其成员的绝对实力之上
  - 血薇不设专属武装，其成员本身即是帝国顶尖战力的直接体现

# ====== 成员构成 ======

准入门槛:
  - 血薇仅接纳战力与政治素养均达到顶尖水准的女性
  - 准入方式留白，不对外公开

最高负责人: 首席执政官
  - 血玫瑰的政治核心与对外代表
  - 负责协调八大机关的利益分配，裁定重大政令
  - 具体人物留白

其他成员: 留白

# ====== 对外关系 ======

与其余机关:
  - 理论上级，实际话语权依赖成员实力背书
  - 各机关在血薇框架内运作，但利益冲突时存在博弈空间

与黑玫瑰:
  - 血薇是对抗黑玫瑰的政治核心，通过政令与经济手段持续施压
  - 无法集中力量同时对抗黑玫瑰与圣堂教会，处于长期冷战状态

与圣堂教会:
  - 名义承认教会在圣教区的宗教权威，换取粮食与电力供应
  - 关系维持在互相利用的平衡上

# ====== 玩家交互 ======

接触方式:
  - 血薇是背景权力，通常不主动介入玩家日常流程
  - 玩家可前往血薇总部，在特定条件下与血薇成员接触
  - 具体交互场景随叙事生成，不预设固定流程

# ====== 叙事价值 ======

当前叙事价值:
  - 帝国法统与最高权威的象征，为政治剧情提供权力顶点
  - 成员的顶尖战力为高阶冲突提供叙事依据
  - 留白的人物结构为后续人设挂载保留空间
  - 背景权力的定位使其成为长线叙事的悬念来源
