<WORLD_机关_蝶翼>
机关名称: 蝶翼

机关分类: 黑玫瑰执行机关，负责军事、执法与情报

建立时间: 新历47年（黑玫瑰成立时整合完成）

# ====== 基本定位 ======

职能定位: 诺丝区唯一武装执法力量，同时承担情报收集与部分黑产武力背书职能。诺丝区无夜莺，蝶翼是该区域的实际治安力量。

规模定位: 介于精锐特种部队与正规军之间。入选门槛极高，以实战能力为核心筛选标准，规模不大但战斗力密度高。

成员构成: 仅接受女性成员，无制度性男性编制。

# ====== 指挥结构 ======

指挥归属: 黑玫瑰顶层直接指挥，蝶翼本身不设独立最高统帅，军事权力不脱离黑玫瑰整体决策体系。

# ====== 驻扎范围 ======

主要驻扎: 诺丝区
延伸存在: 狗镇（交界地），用于维持边境秩序与渗透管控

# ====== 执法逻辑 ======

执法标准:
  - 无正式执法体系，以利益为导向
  - 受害人是有钱人: 优待处理，积极介入
  - 受害人是穷人: 冷处理，基本不管
  - 施害人是穷人: 直接处理
  - 施害人是有钱人: 敲诈后释放

核心原则: 保护黑玫瑰利益，其余事务按利益大小决定介入程度

# ====== 黑产参与 ======

参与方式: 为需要武力背书的黑产业务提供支撑，性质为黑社会武装臂膀

与金丝的关系: 金丝掌握资本与贸易链，蝶翼提供武力背书，顶层由黑玫瑰协调

# ====== 历史脉络 ======

新历16-46年: 作为血薇派驻诺丝区的协防力量，长期驻扎导致与诺丝区利益深度绑定，对血薇不满逐渐积累

新历47年: 被星轨与金丝收买整合，正式加入黑玫瑰

新历48-50年（黑刀之夜）: 清洗内部忠于血薇的派系，蝶翼完成立场转换，成为黑玫瑰核心武装

# ====== 对男性的态度 ======

整体态度: 沿用女尊逻辑，以实力论高下，但不接受男性加入蝶翼编制

个人层面: 成员对男性态度因人而异，无统一标准

# ====== 叙事价值 ======

当前叙事价值:
  - 诺丝区唯一执法力量，为玩家在诺丝区的行动提供压力来源
  - 利益导向的执法逻辑展示黑玫瑰体系的黑社会本质
  - 黑产武力背书为走私、保护费、强制清场等剧情提供制度来源
  - 狗镇驻扎为交界地的黑玫瑰渗透提供具体执行者
  - 不设独立统帅的结构为后续人物设计保留空间
