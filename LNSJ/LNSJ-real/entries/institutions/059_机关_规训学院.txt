<WORLD_机关_规训学院>
机关名称: 规训学院

机关分类: 血玫瑰执行机关，负责教育、驯化与再教育

建立时间: 新历30-35年（随真理部同期成立）

# ====== 基本定位 ======

职能定位: 艾瑞拉帝国的官方教育机构，承担三项核心职能：
  - 男德义务教育: 艾瑞拉区域内男性从幼年至青年的强制义务教育
  - 早泄警犬培训: 警犬上岗前的系统性改造与培训
  - 惩罚再教育: 违法男性、屡教不改者、反抗军成员的强制矫正

核心特征:
  - 多校区分布于艾瑞拉区各处，无统一单一校址
  - 男女学生共用同一校园与设施，但角色截然不同
  - 表面是正规教育机构，内里是系统性驯化体系的核心执行场所

# ====== 教育对象 ======

男性:
  义务教育阶段（幼年至青年，对应幼儿园至高中）:
    - 强制就读，无例外
    - 接受女尊意识形态教育、服从性训练、身体开发
    - 在灵能实练区与女性直接对抗，亲身体验并内化自身的弱势
    - 成绩最差或违反校规的男学生，可被送入调教区由女教官或高年级女性处置

  再教育对象（外部送入）:
    - 违法男性（由夜莺移送）
    - 屡教不改的再犯者
    - 执迷不悟的反抗军成员
    - 警示区候选（表现最差、重大违法、反复反抗者）

女性:
  义务教育阶段:
    - 接受完整的社会学、科学、艺术、灵能等正规学科教育
    - 学习调教技艺，理解如何系统性支配男性
    - 在灵能实练区以男性为对象进行实战训练，体会支配逻辑

# ====== 设施结构 ======

教学区:
  内容:
    - 女尊秩序的理论体系与历史叙事
    - 男性低劣性的"科学依据"（灵核差异、体能差异、社会贡献差异）
    - 服从女性的行为规范与礼仪
    - 女性的完整正规学科教育（社会学、自然科学、艺术、历史）
  授课形式: 男女同堂，男性以被动接受者身份就坐，女性以主动学习者身份参与讨论

灵能实练区:
  用途: 男女直接灵能对抗的实战训练场
  机制:
    - 男女同场对战，无护具保护
    - 男性在生理与灵能层面天然弱于同龄女性
    - 女性通过实战体会"为何男性会被征服"的具体感知
    - 男性通过实战内化"为何自己该被支配"的身体记忆
    - 对男性造成的伤害在合理范围内不追责

调教区:
  用途: 调教技艺的教学与实践场所
  运作规则:
    - 男学生在此学习讨好女性的技巧与身体开发
    - 女学生在此学习系统性调教方法
    - 仅女教官与高年级女学生，可对以下对象进行实质性调教:
      * 从外部送入再教育的违法男性
      * 在校内违反校规的男学生
    - 普通女学生只能观摩或进行轻度练习，不得对再教育对象动用重度手段

警示区:
  用途: 震慑与教育功能并行的展示场所
  展示对象来源:
    - 校内表现最差的男学生
    - 重大违法被移送的男性
    - 屡教不改、执迷不悟的反抗军成员
  展示形态:
    - 活体展示: 被调教、被惩罚的男性以特定状态固定展示
    - 物化展示: 受虐后被物化处理的男性（含肢解形态）
    - 影像记录: 配合活体展示的过程记录
  参观权限: 对外开放，任何女性公民均可入内参观，男性进入须有监管

食堂:
  说明: 男女共用，男性以服务者身份运营，女性以就餐者身份使用

# ====== 早泄警犬培训流程 ======

培训内容:
  - 洗脑调教: 强化对女性的绝对服从意识，消除反抗意识
  - 早泄调节: 通过生理改造与条件反射训练将早泄程度调节至目标等级
  - 服从性调节: 建立"主人命令=绝对真理"的思维模式
  - 抖M性癖培养: 将痛苦与羞辱转化为快感来源
  - 灵能技能培训: 侦查、牵制、追踪相关的灵能应用

入学条件:
  - 灵能感知度与灵能强度达标
  - 付费培训
  - 通过考核后由夜莺分配主人

# ====== 权力结构 ======

归属: 血玫瑰直属机关，与真理部协调配合
运营主体: 女性教官体系，各校区有独立管理层
与夜莺的关系: 夜莺负责将违法男性移送规训学院；警犬培训结束后由夜莺接收分配
与真理部的关系: 课程意识形态内容由真理部审定，规训学院负责执行
