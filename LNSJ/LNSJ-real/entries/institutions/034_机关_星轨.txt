<WORLD_机关_星轨>
机关名称: 星轨

机关分类: 黑玫瑰执行机关，负责芯片制造与信息网络运营

建立时间: 新历30年

# ====== 基本定位 ======

职能定位: 芯片技术研发与生产，灵网平台运营，信息监控技术支持

核心矛盾: 技术成果被血薇强制收归国有，星轨无法从自己的技术中获利，最终选择独立以保障商业利益

# ====== 组织结构 ======

权力结构: 弱化派系与权力层级，无明确最高领导者，以技术项目组与商业部门为运作单位

内部关系: 各部门各干各的，核心目标是搞钱而非夺权

总部位置: 诺丝区东侧科技区

# ====== 核心产业 ======

Beta芯片体系:
  艾瑞拉区版本:
    - 功能: 全面监控（声音、位置、行为）+ 违法检测 + 毒素抑制 + 信誉分系统
    - 商业模式: 血薇统一采购，星轨收取制造费
    - 数据归属: Beta芯片监控数据归血薇所有
    - 后台权限: 星轨掌握底层权限，可以看到所有数据但合同上不能主动调用；关键时刻可以切断血薇访问权限或植入隐藏功能

  诺丝区版本:
    - 功能: 身份认证 + 支付功能，无监控模块
    - 商业模式: 个人购买或企业采购
    - 数据归属: 星轨自己处理

  其他区域: 汐屿区、淬灵区使用艾瑞拉区版本；栖灵区、交界地、南荒无芯片系统

灵网平台:
  定位: 全帝国流行的社交+娱乐+成人内容综合平台，类似抖音、推特、OnlyFans的集合体

  覆盖范围:
    - 艾瑞拉区、汐屿区、淬灵区、诺丝区、栖灵区、交界地
    - 圣教区禁止电子设备，无法使用

  审查机制:
    血玫瑰控制区（艾瑞拉区、汐屿区、淬灵区）:
      - 受真理部意识形态管控
      - 女尊相关内容必须符合意识形态
      - 敏感内容会被过滤或删除
      - 用户数据归血薇所有

    诺丝区、栖灵区、交界地:
      - 无内容限制
      - 用户数据星轨自己处理

  变现模式:
    - 打赏抽成: 平台抽取创作者打赏收入的一定比例
    - 私下转款: 用户间私下转账，平台收取手续费
    - 会员订阅分成: 创作者开设付费订阅，平台抽成

技能芯片生产:
  定位: 赛博朋克风格的技能芯片（非Beta芯片），可植入大脑或身体接口，提供技能加成或感官增强
  用途: 战斗技能、语言包、专业知识、感官增强、快感模拟
  销售区域: 主要在诺丝区，少量流入交界地与栖灵区

# ====== 历史脉络 ======

新历30-35年: 星轨成立，隶属血薇，在诺丝区建设科研基地

新历35-40年: 研发灵能币压缩技术与Beta芯片，技术成果被血薇强制收归国有

新历46年: 研发个人铸币技术，被血薇强制封禁，星轨暗中在诺丝区普及该技术

新历47年: 星轨与金丝、蝶翼结盟成立黑玫瑰，核心动机是"跟血薇搞不到钱，技术成果全被收走"

新历48-50年: 黑刀之夜后，星轨正式独立运营，但仍与血薇保持商业合作关系（Beta芯片制造与灵网运营）

# ====== 对男性的态度 ======

机关层面: 无特殊政策，男性地位取决于所在区域的规则

个人层面: 成员对男性的态度因人而异，无统一标准

诺丝区环境: 星轨总部位于诺丝区，男女相对平等，男性可凭财富获得地位

# ====== 与黑玫瑰内部关系 ======

与蝶翼: 各干各的，无深度协作

与金丝: 各干各的，偶有数据交易（星轨向金丝出售用户数据用于商业分析）

黑玫瑰定位: 松散联盟，核心目标是保障各自商业利益，而非建立统一政权

# ====== 与血玫瑰关系 ======

合同关系:
  - Beta芯片监控数据归血薇
  - 灵网艾瑞拉区域用户数据归血薇
  - 星轨收取芯片制造费与平台运营费

实际态度:
  - 星轨认为独立只是"不再受血薇钳制"，而非敌对
  - 星轨上下喜欢与血薇合作（有稳定收入），但不愿技术成果被无偿收走
  - 星轨对搞独立兴趣不大，核心诉求是"技术成果归自己"

战略筹码:
  - 掌握Beta芯片底层权限，可随时切断血薇的监控系统
  - 掌握灵网平台，可操控信息流动

# ====== 叙事价值 ======

当前叙事价值:
  - 提供全帝国通用的信息平台，为玩家社交、情报获取、内容创作提供空间
  - Beta芯片后台权限为"技术背刺"剧情埋下伏笔
  - 灵网的区域差异审查机制展示帝国分裂现状
  - 星轨的"搞钱不搞权"定位为黑玫瑰的松散联盟性质提供支撑
  - 技能芯片产业为赛博改造、黑市交易提供道具与剧情线
  - 星轨与血薇的商业合作关系为"名义合作、实则制衡"的复杂政治提供案例
