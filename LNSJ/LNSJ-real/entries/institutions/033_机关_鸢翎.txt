
<WORLD_机关_鸢翎>
机关名称: 鸢翎

机关分类: 血玫瑰执行机关，负责医疗保障

建立时间: 新历元年

# ====== 基本定位 ======

职能定位: 帝国医疗体系管理与高端医疗服务，承担全国医疗资源调配、重大疾病治疗、医疗人才培养职能

技术地位: 医疗技术断层级存在，诺丝区分部技术水平显著高于艾瑞拉总部

核心矛盾: 总部（血玫瑰直辖）与分部（诺丝区实控）在运营模式、收入来源、医师待遇上存在巨大差异，血玫瑰正在推进强制迁移，但执行困难

# ====== 组织结构 ======

总部:
  位置: 艾瑞拉
  资金来源: 血薇直接拨款，财政稳定
  运营模式: 公立医院逻辑，医师收入稳定，社会化福利完善
  服务对象: 艾瑞拉权贵女性、官员、重要人物
  设备归属: 所有鸢翎设备归血玫瑰所有，总部可调配

分部:
  位置: 诺丝区
  资金来源: 患者付费+黑玫瑰投资设施
  运营模式: 私立医院逻辑，医师来钱快但竞争激烈
  服务对象: 诺丝区权贵、商人、有钱人
  技术优势: 科技更发达，接触病例更多，医师实战经验丰富
  人员状态: 分部医师不受总部强制约束，可自由选择去留

# ====== 总部与分部的差异 ======

医师待遇对比:
  总部:
    - 血薇直接拨款，收入稳定但不高
    - 社会化福利完善：住房分配、子女教育、退休保障
    - 工作压力相对较小，患者多为权贵，医患关系相对和谐
    - 晋升路径清晰，与血玫瑰体系深度绑定
    - 社会地位高，被视为"国家医师"

  分部:
    - 患者付费，收入高但波动大
    - 社会化福利缺失，一切靠自己
    - 工作压力大，竞争激烈，需要不断提升技术保持竞争力
    - 晋升路径不明确，依赖个人声誉与患者资源
    - 社会地位取决于个人能力与财富积累

对女性患者的服务质量:
  总部与分部一致: 医疗质量有保证，技术过硬，服务态度良好，治疗方案规范

对男性患者的处理方式:
  总部（艾瑞拉）:
    - Beta芯片监控: 下毒会被检测到，风险高
    - 男性是血玫瑰财产: 男性的灵能币回流国库，私自下毒损害国有资产
    - 医师收入稳定: 没必要冒险，血薇拨款已经够用
    - 结果: 总部相对"道德"，不是因为医德高尚，而是风险高+收益低

  分部（诺丝区）:
    - 无Beta芯片监控: 下毒不会被系统检测，风险低
    - 男性不属于国家财产: 男性产出价值与鸢翎医师无关，死活不影响收入
    - 私药与假账: 开私药、做假账，钱进私人腰包而非公账（公账会被血玫瑰收走收益）
    - 逐利压力: 来钱快但竞争大，医师为了收入主动下毒让患者反复生病
    - 常见手段:
      * 在药物中添加成瘾性成分，患者对该医师产生依赖
      * 添加低剂量慢性毒素，当前病治好，数月后引发其他病症
      * 故意开不完全对症的药，延长治疗周期
      * 手术时故意留隐患，数年后需要二次手术
    - 结果: 分部"黑心"，不是因为道德败坏，而是风险低+收益高

制度层面:
  - 没有明确的"对男性下毒"政策
  - 但分部环境（无监管+逐利压力）自然导致这种行为成为常态
  - 血玫瑰高层知道但管不了（因为分部在诺丝区，黑玫瑰地盘）
  - 总部医师偶尔也有下毒行为,但比例远低于分部，且更隐蔽

# ====== 迁移冲突 ======

血玫瑰的迁移命令:
  目标: 调医生回艾瑞拉，迁移设备，将医疗资源集中到总部
  执行方式:
    - 下达强制调令，要求分部医师回艾瑞拉总部报到
    - 迁移分部设备（设备属于血玫瑰，想拿走就拿走）
    - 不回去就辞职，不想辞职就得听从调配
  目的:
    - 削弱诺丝区分部的技术优势
    - 将核心医疗人才集中到血玫瑰控制区
    - 防止鸢翎分部进一步脱离掌控

医师的选择逻辑:
  回总部的医师:
    - 看重稳定收入与社会化福利
    - 不愿承受诺丝区的竞争压力
    - 家庭在艾瑞拉，不愿分离
    - 政治忠诚度高，认同血玫瑰体系

  留分部/辞职当私医的医师:
    - 看重高收入，愿意承受竞争压力
    - 在诺丝区已有稳定患者资源与人脉
    - 享受诺丝区的自由度与娱乐产业
    - 不愿受血玫瑰体系束缚
    - 这是纯粹的个人利益权衡，与政治立场无关

迁移进度:
  当前状态: 正在执行中，但进展缓慢
  困难:
    - 大量核心医师选择辞职而非回总部
    - 设备迁移后分部医疗能力下降，但黑玫瑰持续投资新设备
    - 总部无法强制医师回去，只能依靠调令与福利吸引
  结果: 血玫瑰迁走了设备，但人才流失严重；黑玫瑰投资新设备，分部医疗能力恢复

黑玫瑰的态度:
  策略: 不主动拉拢医师，只投资设施
  逻辑:
    - 自然有医师为了钱留下，黑玫瑰坐收渔利
    - 投资新设备，保持诺丝区医疗水平
    - 不与血玫瑰在政治层面直接冲突，只在经济层面竞争

# ====== 医师出走现象 ======

出走规模: 迁移命令下达后，分部约40%的医师选择辞职成为私人医生

出走原因:
  - 收入更高: 私人医生直接为诺丝区权贵、金丝高层、蝶翼军官服务，收入远超总部
  - 自由度更大: 不受鸢翎体系约束，工作时间与地点自主
  - 患者资源稳定: 在诺丝区已建立口碑与人脉，不缺患者

出走医师的服务对象:
  - 诺丝区权贵女性
  - 金丝高层
  - 蝶翼军官
  - 有钱的男性（收费更高，下毒更狠）

与黑玫瑰的关系:
  - 这些医师不属于黑玫瑰，但客观上为黑玫瑰服务
  - 黑玫瑰通过金丝为这些私人医生提供设备支持与患者介绍
  - 医师与黑玫瑰是互利关系，不是雇佣关系

# ====== 权力结构 ======

最高负责人:
  位置: 艾瑞拉总部
  立场: 血玫瑰派系
  职责: 执行血玫瑰的医疗政策，协调总部与分部关系，推进迁移命令
  当前困境: 迁移命令执行困难，人才流失严重，总部技术优势无法恢复

内部关系:
  - 总部医师对分部医师: 羡慕其收入，但不愿承受竞争压力
  - 分部医师对总部医师: 蔑视其"吃死工资"，认为总部医师技术停滞
  - 出走医师对体系内医师: 完全独立，不再关心鸢翎内部事务

# ====== 叙事价值 ======

当前叙事价值:
  - 展示血玫瑰-黑玫瑰对峙在医疗领域的具体表现
  - 提供医疗系统中"公立vs私立""中央vs地方"冲突的具体案例
  - 为玩家提供在艾瑞拉与诺丝区就医的不同体验
  - 总部的"相对道德"与分部的"黑心逐利"形成对比，展示制度环境对行为的影响
  - 医师出走现象为"人才流失""忠诚与利益的权衡"提供叙事素材
  - 男性在分部被反复下毒的遭遇为玩家提供"被医疗系统压榨"的体验
  - 迁移冲突为血玫瑰"有名无实的控制力"提供具体体现
  - 为玩家提供攻略路线：在艾瑞拉享受稳定医疗，在诺丝区警惕黑心医师，或成为私人医生的客户/合作者
