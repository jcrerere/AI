机关名称: 真理部

机关分类: 血玫瑰意识形态机关，负责宣传、审定、档案与舆情控制

# ====== 基本定位 ======

职能定位: 艾瑞拉帝国最核心的意识形态机关，负责定义“什么是女尊秩序”、什么样的内容可以传播、哪些人需要被改造为样板或反面教材。真理部不直接负责街面执法，但它为夜莺、规训学院与各区行政机关提供合法性叙事与宣传模板。

核心职能:
  - 宣传审定: 审核全帝国公开传播的教材、影像、广播、偶像包装与城市宣传口径
  - 舆情引导: 监测区域舆论、谣言与集体情绪波动，必要时下发统一口径与封禁令
  - 档案封存: 建立“样板档案”“负面教材档案”“高风险男性档案”，供其他机关引用
  - 心理矫正: 对严重偏离女尊秩序的男性与异端对象提供思想矫正与记忆重塑方案
  - 机构校准: 对规训学院课程、夜莺宣讲内容、官方偶像叙事与重要审判案例给出宣传定性

# ====== 权力结构 ======

核心设施:
  - 真理部大楼: 艾瑞拉中心区的总部大楼，承担审定、封存、问询与样板制作功能
  - 档案层: 保存宣传样板、负面教材、敏感案例与跨机构调卷记录
  - 审定厅: 各类教材、公开内容、广告与偶像包装在此定稿
  - 调教宣传室: 将惩戒、月结、公开处置包装为“秩序教育”的专用空间

内部结构:
  - 审定官: 负责公开内容与教材的最终定性
  - 舆情观察员: 监测城市广播、灵网热点、街区情绪与重大流言
  - 档案管理员: 维护封存记录与跨机关调阅权限
  - 样板编修组: 把真实事件改写为可供全帝国传播的秩序故事

# ====== 对外关系 ======

与血薇:
  - 真理部为血薇提供意识形态正当性，是血玫瑰维持法统叙事的重要支柱

与规训学院:
  - 规训学院负责执行，真理部负责定义课程方向、惩戒范式与女尊叙事模板

与夜莺:
  - 夜莺完成街面控制与税务执行后，重要案例会被真理部加工成“秩序宣传样板”

与审判庭:
  - 审判庭维持法律裁决独立性，但重大案件的社会解释权常被真理部掌握

与星轨:
  - 真理部依赖星轨的监控与内容通道分发宣传，同时警惕诺丝区的信息反渗透

# ====== 玩家接触面 ======

常见接触方式:
  - 被列入宣传样板、负面教材或敏感档案
  - 在真理部大楼、审定厅、档案层或调教宣传室中接受问询
  - 因舆情事件、月结异常、公开惩戒或高热灵网内容被真理部盯上
  - 通过潜入或交易接触被封存的宣传稿、黑档案与样板素材

叙事用途:
  - 将“法律”“税务”“惩戒”“偶像文化”统一到一套意识形态框架下
  - 让玩家感到艾瑞拉的压迫不只是暴力，还有对现实解释权的垄断
  - 为潜入档案层、篡改样板、删除黑档案、制造假舆情提供目标机关
