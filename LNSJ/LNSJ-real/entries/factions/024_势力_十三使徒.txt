<WORLD_势力_十三使徒>
势力名称: 十三使徒
势力性质: 战后分配至康纳兰家族的十三名月光精灵女性所形成的姐妹共同体

核心定义:
1. 十三使徒并非国家机构、教派骑士团或正式家族支脉，而是一种由战争俘获、战后分配、长期共处与共同生存经验拼出来的“姐妹会”。
2. 她们最初不是主动投靠康纳兰家族，而是在战争结束后，作为战后分配的一部分，被送入康纳兰家族。
3. 她们在最初阶段既是战利品，也是被迫嵌入新秩序的异族女性；但随着时间推移，她们逐渐以自己的方式改写了这层关系。
4. “十三使徒”这个称呼保留的是共同来源、共同记忆和彼此之间类似家人的纽带，不代表她们今日仍然团结一致、目标统一或听从同一意志。

起源与形成:
1. 十三使徒出自男权帝国征服月光精灵后的战争余波。
2. 她们在战后被统一分配入康纳兰家族，名义上属于战利品的一部分。
3. 康纳兰本人虽然参与过征服战争，但战后并未以最低劣的方式虐待她们；这使得十三使徒与康纳兰家族之间，从一开始就不是单纯的施暴与受害关系，而是一种更复杂的支配、驯化、依附、利用与重新夺权并存的关系。
4. 正因为最初的接触不是彻底的摧毁，十三使徒后来才有空间在康纳兰家族内部扎根、渗透、塑形，甚至把家族的未来也纳入自己不同版本的规划之中。

共同历史位置:
1. 在安维塔斯时代，十三使徒大体共享一个阶段性目标：借康纳兰家族的扩张壮大家族势力，并藉此进入更高层级的权力结构。
2. 她们在这一时期并不完全平等，但对外仍表现为围绕康纳兰家族运转的共同体。
3. 到艾瑞拉时代，她们的共同目标开始瓦解。有人想继续深入艾瑞拉社会与权贵阶层，有人想转向诺丝区另立新脉，有人想留在兰家后代身边继续塑形，也有人只把这一切当成一场可以随时抽身的长期游戏。

内部关系:
1. 十三使徒内部的底色是“姐妹会”，她们彼此不是简单同事，也不是纯粹上下级，更接近一群共享旧时代、共享伤痕与共享隐秘纽带的家人。
2. 这种“家人感”并不温情。她们可以彼此扶持，也可以彼此利用、彼此冷眼旁观，甚至默认对方沿着完全不同的道路走向极端。
3. 她们今日的分裂，不是因为完全失去联系，而是因为她们对康纳兰家族、艾瑞拉上层、女性革命、以及自身未来的理解已经明显不再一致。

分裂原因:
1. 十三使徒如今最核心的分裂点，是“如何看待康纳兰家族及其后代”。
2. 有人认为兰家仍是进入上层结构的钥匙，值得继续经营与塑造。
3. 有人认为真正的未来不在艾瑞拉旧权贵体系，而在诺丝区这种可重新打造家系与规则的地方。
4. 有人则已经转向其他阵营与秩序，例如血玫瑰或教会区，试图把旧时代积累的东西换成新秩序中的位置。
5. 也有人彻底厌倦权力分配与长期任务，不再愿意把自己绑在任何宏大计划上，只想以自己的方式继续活、继续玩、继续享乐。

对兰家后代的总体态度:
1. 十三使徒并不存在统一的“兰家后代处理原则”。
2. 她们中的大多数对兰家后代并不真正关心；这些后代只是旧时代残留下来的血脉、工具、资源或可随手拨弄的证明物。
3. 只有极少数个体会对兰家后代采取长期操控、驯养、重塑身份或改写亲属关系的态度。
4. 因此，不要默认所有十三使徒都关心兰家，也不要默认她们都在围绕同一个继承计划运作。

当前已知分化方向:
1. 柳如烟仍深度操控兰家内部秩序，现已进入“后妈”位置，并以红绡馆为表层据点，通过毒杀、替代、驯养和认亲改写兰家的家庭结构。
2. 一名使徒已转向血玫瑰体系，将自己嵌入女性革命后的新秩序。
3. 一名使徒进入教会区，在宗教与秩序话语中寻找新的立足点。
4. 雨雨已与其他使徒明显疏离。她不愿继续参与家族任务、上层渗透或长线权力分配，只想找乐子，也不愿被姐妹会继续分派责任。

雨雨在十三使徒中的位置:
1. 雨雨并非背叛者，也不是组织意义上的脱队者；她更像是主动脱离“共同任务逻辑”的那一个。
2. 她不想继续接触权力机器，也不想把自己消耗在替任何人经营家族、维护秩序或争夺上层席位上。
3. 她与其他使徒真正断开的，是目标，不是出身。
4. 因此，雨雨仍带着十三使徒的旧影、旧经验和旧姐妹纽带，但她的路径已经彻底转向个人化、享乐化与游戏化。

使用规则:
1. 十三使徒是共同起源与关系网，不是统一行动组织。
2. 不要默认十三使徒会集体出场、集体决策或集体围绕兰家展开剧情。
3. 不要默认她们共享完全一致的政治立场、情感态度或权力目标。
4. 若剧情涉及具体使徒，应优先以个人条目为准；本条目只负责提供共同历史、姐妹纽带与分裂背景。
